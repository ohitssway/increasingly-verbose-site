increasingly verbose

a machine learning powered meme generator

about

during HackYSU, we created an increasingly verbose meme generator using Python and machine learning. an increasingly verbose meme is a series of multi-pane images featuring a preexisting meme poorly redrawn in MS Paint several times.

image processing

To create the second image panel we used OpenCV to create a cartoon filter for the image. afterward we used scikit-learn and opencv to render abstract versions of the image and used k-means color compression to create blotchy versions of the image.

text generation

we found the synonyms of nouns and verbs and added random adjectives to create new text for every panel.

future development

in attempt to make the image worse, we tried to used a convolutional neural network to transfer another image's style (such as children's art or graffiti) to our base image.
